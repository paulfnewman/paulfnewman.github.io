About

Hi, I'm Paul Newman, front-end web developer, Staffordshire University graduate, husband and dog owner.

I hold a Bachelor of Science First-Class Honours degree in Computing Science and a HND Distinction in Applied Computing.

Before taking up web design I was a paramedic and, before that, a sergeant with Warwickshire Police, but after nearly 20 years in the emergency services I decided to change career.

Now, I design websites.  I'm hard-working, highly motivated and passionate about all things web-related.

Recent Work

Ambo Aid

Ambo Aid is a web app which was designed and built to fulfil the requirements of a BSc Computing Science course module.
It is a mock-up of an application which could potentially be used by patients and ambulance clinicians and which would provide important information in medical emergencies. The app makes extensive use of PHP and MySQL to read from and write to a secure rear-end database.

The Chase Golf, Health Club and Spa
- Website Update


The website for The Chase Golf, Health Club and Spa was created by inLife Design, a web design and digital marketing company based in Burntwood, Staffordshire.
Whilst working for inLife, the entire Spa Treatments section of the website was restyled in accordance with the client's desire to affect wide-ranging modifications.

Woodlands Restaurant
- Website Update


The Woodlands Restaurant website was also created by inLife Design.  Subsequent updates to the menus on the site were made on behalf of inLife, in accordance with the client's wishes.

The Lone Twin Network Database

The Lone Twin Network is a UK-based voluntary support group, run by and for adult lone twins who have lost their sibling at any life stage.  Wishing to modernise their records, the group wanted a digital membership database.  To meet their needs, a user-friendly Microsoft Access relational database was produced, to store the details of their 500-plus members from all over the world.

Olde Money

Olde Money is an educational website which aims to teach school children about British pre-decimal currency, and was built for a Staffordshire University HND course module.  Designed for children between the ages of 7 and 11, the website contains child-friendly images and interactive quizzes.
Built using the popular Bootstrap web framework, the site makes extensive use of animate css, resulting in a highly dynamic user experience.

Benson's Space

Another website built for a university HND course module, Benson's Space is a website dedicated to the issue of pet security.
Once again, it was built using the fully-responsive Bootstrap front-end web framework, which means that the site can be easily viewed on a variety of different device sizes.  All videos which appear on the site were self-made.

Botany Bay Bulletin

When a proposed housing development threatened the small rural community of Botany Bay near Lichfield, a website was produced to act as an information portal for local residents.   The site was created with the easy-to-use WordPress content management system, which allows a website administrator with no technical knowledge of web design or coding to log-in and update website content.
